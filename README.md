## 运行

1. 克隆仓库

```sh
git clone https://github.com/F-loat/cards-ahoy.git
```

2. 安装依赖

```sh
pnpm install
```

3. 运行项目

```sh
pnpm dev
```

4. 预览效果

使用微信开发者工具导入项目
